Haskell libraries for Computer Vision

This is an experimental Haskell framework for fast prototyping of computer vision and image processing applications. We take advantage of the expressive power of functional programming to easily write elegant programs, supported by standard low level specialized libraries.

Disclaimer: This software is currently extremely provisional and unstable. It may not work "out of the box" and I do not recommend it for serious applications.

Requirements

Intel IPP, mplayer, and several standard Haskell packages available from Hackage.

Source Code

darcs get http://code.haskell.org/easyVision

Installation

The system is not yet in Hackage but the packages are in standard cabal format and can be easily installed.

See the INSTALL file for details.
